This is a community-driven, open source wiki for frequently asked questions about Riftbound, a trading card game. It's designed to help judges and players quickly find answers to common rules questions and gameplay scenarios.

## Purpose

When players ask repeated questions in community channels, judges can link directly to the relevant pages here instead of explaining the same rules multiple times. This ensures consistent, accurate information and helps the community learn the game more effectively.

## Contributing

This wiki is an open source project maintained by the Riftbound community. If you'd like to contribute new content, fix errors, or improve existing pages, contributions are welcome!

## Using This Wiki

Browse the pages to find answers to specific rules questions, card interactions, and gameplay scenarios. Each page focuses on a particular topic or frequently asked question to make information easy to find and share.
